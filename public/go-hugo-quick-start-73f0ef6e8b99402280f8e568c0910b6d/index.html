Hugo 설치

Ubuntu

1

sudo apt-get install hugo

Windows

릴리즈 버전만 깃허브에서 다운 받아서 C:\Hugo\bin에 넣어주고 환경변수 셋팅만 해주면 된다.

Hugo new

new 키워드는 새로운 프로젝트를 만들때 사용할 수 있고,

1

hugo new site quickstart

새로운 포스트를 만들때도 사용할 수 있다.

1

hugo new posts/aa
